Detailed Description

Widget for entering diet-related information including age, sex, weight, and height. @inherits QWidget.

The InputWidget class is designed to allow users to input information such as age, sex, weight, and height for creating a new Diet object. It emits signals for changes in weight and height units, as well as when the user submits the entered information.
