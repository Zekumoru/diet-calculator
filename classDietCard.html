Detailed Description

Widget representing a diet card with age, sex, weight, height, BMI, and BMR information. @inherits QWidget.

The DietCard class is a widget designed to display information related to a Diet object, including age, sex, weight, height, and the calculated BMI and BMR. It provides slots for handling unit changes and emits a signal when the user clicks on the delete button.
